# Portfólio de Testes Manuais – Gustavo Gama

Este repositório contém casos de teste manuais elaborados para diferentes aplicações web de demonstração, com foco em prática de QA.

## Casos de teste incluídos

1. **TC-001 – Login com credenciais válidas (SauceDemo)**
   - Aplicação de e-commerce de demonstração.
   - URL: https://www.saucedemo.com
   - [Ver caso de teste](casos-de-teste/tc-001-saucedemo-login.md)

2. **TC-002 – Adicionar novo registro na Web Table (DemoQA)**
   - Aplicação de componentes de UI para testes.
   - URL: https://demoqa.com/webtables
   - [Ver caso de teste](casos-de-teste/tc-002-demoqa-webtables-add-record.md)

3. **TC-003 – Adicionar e remover elementos (The Internet – Herokuapp)**
   - Aplicação de demonstração para testes de QA.
   - URL: https://the-internet.herokuapp.com/add_remove_elements/
   - [Ver caso de teste](casos-de-teste/tc-003-the-internet-add-remove-elements.md)

---

Todos os testes são **manuais**, executados diretamente no navegador, seguindo boas práticas de QA funcional.
