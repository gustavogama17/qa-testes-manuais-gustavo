# Caso de Teste TC-003 – Adicionar e remover elementos (The Internet – Add/Remove Elements)

**Aplicação:** The Internet – Add/Remove Elements  
**URL:** https://the-internet.herokuapp.com/add_remove_elements/  
**Tipo de teste:** Funcional – Manual  
**Objetivo:** Verificar se o sistema permite adicionar múltiplos elementos e removê-los corretamente.

---

## Pré-condições

- Ter acesso à internet.
- Navegador aberto.
- Site https://the-internet.herokuapp.com/add_remove_elements/ disponível.

---

## Passos para execução

1. Acessar a URL `https://the-internet.herokuapp.com/add_remove_elements/`.
2. Localizar o botão **Add Element**.
3. Clicar no botão **Add Element** **três vezes**.
4. Contar quantos botões **Delete** aparecem na tela.
5. Clicar no primeiro botão **Delete** exibido.
6. Contar novamente quantos botões **Delete** restaram na tela.

---

## Dados de teste

- Não há dados de entrada (apenas interação com os botões).

---

## Resultado esperado

- Após o passo 3, devem existir **3 botões "Delete"** na tela.
- Após o passo 5 (remover um elemento), devem restar **2 botões "Delete"** na tela.
- Nenhuma mensagem de erro deve ser exibida.
- A página não deve recarregar sozinha.
